# stockcheck 리빌드 설계 스펙: 초간편 주식 판결 & 손실방어 서비스

**날짜:** 2026-04-01
**상태:** 승인됨

## 요약

기존 "검색 → 체크리스트 3문항 → 카드 스와이프" 플로우를 "검색 → 금액 입력 → 판결 → 시뮬레이션" 3단계로 단순화.
타겟: FOMO에 눈먼 주식 왕초보. 핵심 질문: "그래서 사? 말아?"

---

## 플로우

```
로그인 → 검색(종목명) → 금액 입력 → [로딩] → 판결 결과(스크롤) → "그래도 사면?" → 시뮬레이션
```

## 핵심 기획 결정

| 항목 | 결정 |
|------|------|
| 로그인 | 현행 유지 (카카오/구글 OAuth) |
| 대상 | 한국 주식만 (MVP) |
| AI | Gemini API (추후 Claude 교체 대비 모듈화) |
| 뉴스 | 네이버 뉴스 검색 API |
| 판결 | 4단계: 절대금지🚨 / 대기🤔 / 괜찮아 보여👀 / 관망🫥 |
| 투자금 | 유저 입력 (기본값 100만원) |
| 결과 | 한 화면 스크롤 (카드 스와이프 X) |
| 근거 | 비유 메인 + info 아이콘으로 용어 설명 |
| 시뮬레이션 | 단일 예측 (5/10/20년), 치환 메인("테슬라 주인이야") + 금액 서브 + 미니 그래프 |
| 디자인 | 토스 플랫 + 판결 영역만 iOS 글래스모피즘 + 스티커 이모지 + 이모지 색 블리딩 |
| 히스토리 | 판결 히스토리만 |
| 제거 | 체크리스트, Swiper, 가상매수, 카카오 공유 |

## 판단 로직 (100점 스코어링)

### 3축 구조

| 축 | 배점 | 지표 |
|----|------|------|
| 종목 | 40점 | 가격 위치(12) + PER 밸류(10) + 거래량(10) + 수급(8) |
| 시장 | 40점 | VIX(10) + 공포탐욕(8) + 코스피/코스닥 위치(8) + 금리(8) + 달러/유동성(6) |
| 뉴스 | 20점 | 루머 비율(12) + 반영 여부(8) |

높을수록 위험. >= 70: 절대금지, >= 45: 대기, < 45: 괜찮아 보여, 데이터 부족: 관망.

### 동적 조정
- 종목 고점(>=90%) + 과열(거래량>=3배) 동시 → 종목 점수 * 1.2
- 시장 5축 중 3개+ 동시 위험 → 시장 점수 * 1.3

### 강제 트리거
- 52주 고점 98%+ AND 거래량 5배+ → 무조건 절대금지
- VIX >= 35 → 무조건 절대금지
- 기관+외인 5일 연속 동시 순매도 → 무조건 절대금지

### 데이터 소스
- KIS API: 시세, 재무, 일별 시세, 기관/외인 수급
- assetx2-dashboard API: /api/yahoo (VIX, 코스피/코스닥), /api/fred (금리), /api/dxy (달러)
- 네이버 뉴스 검색 API: 최근 10개 뉴스

## 디자인

- 기본: 토스 플랫 (기존 TDS 토큰 유지)
- 판결 영역: iOS 글래스모피즘 (backdrop-filter blur + 반투명 배경)
- 스티커 이모지: 흰 테두리 + drop-shadow + float 애니메이션 (CSS만, 이미지 불필요)
- 이모지 색 블리딩: 이모지 자체 색이 글래스 배경에 radial-gradient로 번짐
- 판결 등급 이모지: 🚨(강한톤) 🤔(유머) 👀(유머) 🫥(중립)
