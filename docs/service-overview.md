# stockcheck 서비스 개요

## 한 줄 정의

> "그래서 사? 말아?" — FOMO에 눈먼 초보를 위한 초간편 주식 판결 & 손실방어 서비스

종목 추천이나 수익 예측 앱이 **아님**.
"지금 들어가도 되는지" AI가 판결을 내려주고, 손실/수익을 실생활로 치환해서 보여주는 서비스.

---

## 타겟 사용자

- 2~30대 주식 왕초보
- 숫자가 공포이고, PER/PBR 같은 전문 용어가 외계어인 사람
- "그래서 지금 사? 말아?" 이 질문 하나만 머릿속에 있는 사람
- FOMO에 눈 돌아가서 충동 매수 직전인 사람

---

## 핵심 가치

| 기존 서비스 | stockcheck |
|------------|-----------|
| 복잡한 차트·지표 | "사지 마!" / "괜찮아 보여" 한마디 |
| 전문 용어 나열 | "치킨 5마리 날릴 확률 85%" 실생활 치환 |
| 숫자 공포 | 비유로 설명, 용어는 info 아이콘 뒤에 숨김 |
| 판단은 유저 몫 | AI가 판결 내려줌 (4단계) |

---

## 서비스 플로우

```
[로그인] → [종목 검색] → [투자 금액 입력] → [AI 판결] → [시뮬레이션]
```

### 1단계 — 로그인
- 카카오 OAuth / Google OAuth
- 히스토리 저장을 위해 로그인 필수

### 2단계 — 종목 검색
- "어떤 주식에 눈 돌아갔어?" 타이틀
- 종목명 또는 6자리 코드 입력
- 인기종목 칩 (최대 5개) — 탭하면 바로 검색

### 3단계 — 투자 금액 입력
- "얼마나 넣을 생각이야?"
- 만원 단위 숫자 입력 (기본값 100만원)
- 건너뛰기 가능 (기본값 적용)

### 4단계 — AI 판결
Edge Function이 3개 API 병렬 호출 → 3축 스코어링 → 판결 등급 결정 → Gemini가 멘트 생성

**3축 스코어링 (100점 만점, 높을수록 위험):**

| 축 | 배점 | 데이터 소스 |
|----|------|------------|
| 종목 | 40점 | KIS API (가격 위치, PER, 거래량, 수급) |
| 시장 | 40점 | assetx2 API (VIX, 공포탐욕, 코스피/코스닥, 금리, 달러) |
| 뉴스 | 20점 | 네이버 뉴스 (루머 비율, 반영 여부) |

**판결 4단계:**

| 등급 | 이모지 | 예시 멘트 |
|------|--------|----------|
| 절대금지 | 🚨 | "사지 마! 지금 사면 넌 기부 천사야" |
| 대기 | 🤔 | "3일만 참아. 지금은 좀 애매해" |
| 괜찮아 보여 | 👀 | "나쁘지 않은데? 몰빵은 하지 마" |
| 관망 | 🫥 | "데이터 부족. 판단 보류" |

**결과 화면 (한 화면 스크롤):**
- 판결 이모지 + 뼈때리는 한줄 멘트 (글래스모피즘 카드)
- 손실/수익 치환: "치킨 5마리 날릴 확률 85%"
- 초보용 근거: 비유 메인 + info 아이콘 탭 시 한줄 용어 설명
- "그래도 사면?" 버튼 → 시뮬레이션으로 이동
  - 절대금지 판결: "그래도 볼 거야? 진짜?" 재확인

### 5단계 — 시뮬레이션
"그래도 사면?" 클릭 시 5/10/20년 장기 예측 표시:

```
5년 뒤,  넌 에어팟 프로 주인이야 🎧
         약 440만원

10년 뒤, 넌 유럽여행 다녀온 사람이야 ✈️
         약 650만원

20년 뒤, 넌 테슬라 모델3 주인이야 🚗
         약 1,400만원
```

- 치환 아이템이 메인, 금액은 서브
- 미니 라인 차트 (성장 곡선)
- 과거 연평균 수익률 기반 단일 예측

## 디자인 컨셉

- **기본**: 토스 플랫 (TDS 기반 디자인 시스템)
- **판결 영역**: iOS 글래스모피즘 (backdrop-filter blur)
- **스티커 이모지**: 흰 테두리 + 그림자 + float 애니메이션 (CSS만, 이미지 불필요)
- **이모지 색 블리딩**: 이모지 자체 색이 글래스 배경에 연하게 번짐
- 참고: JOMO 앱 스타일

## 기술 스택

| 구분 | 기술 |
|------|------|
| 프론트엔드 | Vite 6 + React 19 + Tailwind CSS v4 |
| 인증 | Supabase Auth (카카오/구글 OAuth) |
| DB | Supabase PostgreSQL |
| 서버 로직 | Supabase Edge Function (Deno) |
| 주식 데이터 | KIS (한국투자증권) Open API |
| 시장 데이터 | assetx2-dashboard API (VIX, 금리, 코스피/코스닥) |
| 뉴스 | 네이버 뉴스 검색 API |
| AI 분석 | Gemini API (추후 Claude 교체 대비 모듈화) |
| 배포 | Vercel (프론트) + Supabase (Edge Function) |

## DB 스키마
